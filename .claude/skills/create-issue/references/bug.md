# バグ報告 (bug)

## 収集する情報

AskUserQuestion で以下を対話的に聞き出す。
ユーザーの発言から推測できる項目はスキップし、足りない部分だけを聞く。

1. **何が起きたか** — 観測された挙動（必須）
2. **期待される挙動** — 本来どうあるべきか（必須）
3. **再現手順** — 再現できる場合のみ（任意）

## 深掘り

コードベースを調査し、原因の仮説や影響範囲を簡潔に共有する。
ユーザーに追加情報がないか確認する。

## タイトル

「〜できない」「〜が誤っている」など、症状を簡潔に表す。
例: 「年代スライダーで紀元前の年が正しく表示されない」

## テンプレート

```markdown
## 現象

[何が起きたか]

## 期待される挙動

[期待される挙動]

## 再現手順

1. [手順]

## メモ

- [追加情報]

---
🤖 Generated with [Claude Code](https://claude.com/claude-code)
```

再現手順がない場合は「## 再現手順」セクションごと省略する。
メモがない場合は「## メモ」セクションごと省略する。

## ラベル

- ラベル名: `bug`
- 色: `D73A4A`
- 説明: `Something isn't working`
