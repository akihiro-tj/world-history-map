# 領土選定ガイド

## 選定基準

1. **教科書基準**: [data-quality.md](./data-quality.md) の「知識レベル」に準拠する国家・帝国・文明
2. **地域バランス**: ヨーロッパに偏らず、アジア・中東・アフリカ・南北アメリカをカバーする
3. **時代カバレッジ**: 古代（-500）から現代（2000）まで各時代を網羅する

## 優先度の定義

| 優先度 | 基準 | 目安 |
|---|---|---|
| P1 | 教科書の柱となる帝国・王朝。どの教科書にも必ず登場し、章の主題になる | ローマ帝国、唐、オスマン帝国、フランス |
| P2 | 主要な章で扱われる国家。大学入試（共通テスト・国公立二次）でも頻出 | ムガル帝国、プロイセン、神聖ローマ帝国 |
| P3 | 教科書に登場するが、やや補足的な位置づけ。地域史の文脈で言及される | ガーナ王国、マリ帝国、サファヴィー朝 |
| P4 | 教科書に名前は出るが詳しい記述は少ない。発展的学習の範囲 | マジャパヒト王国、ヴィジャヤナガル王国 |

データ作成時は P1 → P2 → P3 → P4 の順に取り組む。同じ優先度内では時代順（古代→近現代）で処理する。

## 除外カテゴリ

- 先住民族・部族領土（例: オーストラリア先住民族、北米先住民族）
- 植民地の細分化（例: "Cuba (Spain)"、"Ceylon (Dutch)"）
- 教科書に登場しない小規模な都市国家
- 十分な歴史データがない領土
