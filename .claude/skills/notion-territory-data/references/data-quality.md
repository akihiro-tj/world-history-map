# データ品質ガイドライン

## 知識レベル

日本の高校世界史教科書（山川出版社『詳説世界史』）に登場する水準の情報を基準とする。

### 取り上げるべき情報
- 高校世界史で一般的に学習する国家・王朝・人物・事件
- 世界史の重要語句として広く認知されている用語・概念
- 大学入試（共通テスト・国公立二次）で問われる程度の知識

### 避けるべき情報
- 専門的な学術論文でしか扱われない詳細（例: 特定の条約の細かい条文内容）
- 高校世界史の範囲を超えるマイナーな人物・事件・地名
- 研究者間で定説が確立していない学説や解釈

### 判断に迷った場合
- 「高校生が世界史の授業で学ぶか？」を基準にする
- 複数の教科書で共通して取り上げられている事項かどうかを目安とする

## 記述言語

すべてのコンテンツは日本語で記述する: profile の値、context テキスト、key_events の説明。

## 禁止値

どのフィールドにも "不明" を使用してはならない。情報が不明な場合はフィールドごと省略する。

## context フィールド

### 要件
- 2〜3 文、50〜200 字（厳密なバリデーション）
- 客観的事実のみ: 日付、イベント名、定量的な状態、因果関係・時間的前後関係
- 選択年に固有: その年におけるその領土の状況を記述する
- 高校世界史で一般的に扱われる水準の情報に留める

### 良い例
未来の文脈で補足するパターン:
```
1700年のフランスはルイ14世の親政期にあり、ヨーロッパ最大の人口と軍事力を擁していた。翌1701年にはスペイン継承戦争が勃発する。
```

過去の文脈で補足するパターン:
```
800年のフランク王国はカール大帝のもとで西ヨーロッパの大部分を統一していた。この年、カール大帝はローマ教皇レオ3世から西ローマ皇帝の帝冠を受けた。
```

### 悪い例
- "フランスは美しい国であった" → 主観的評価
- "フランスは世界史上最も重要な国の一つである" → 文学的修辞
- "首都はパリで政体は絶対王政であった" → profile フィールドとの重複
- "1700年のフランスではコルベール主義の残滓としてマニュファクチュール王立監督制度が…" → 教科書水準を超えた専門的詳細

### 執筆パターン
1. **その年の状態を述べる**（必須）: 誰が統治し、どのような状況にあったか（例: 最盛期、衰退期、戦時中）
2. **前後の文脈で補足する**（1〜2文目の残りで）: 直前に何が起きたか、または直後に何が起きるかを添えて時代の流れを示す

50〜200字の制約があるため、1 を優先し、2 は字数に余裕がある範囲で加える。過去と未来の両方を入れる必要はなく、その年の状況を理解するのに役立つ方を選ぶ。

## key_events フィールド

### 要件
- 1 領土あたり最低 3 件のイベント
- 年の昇順でソート
- 領土の歴史全体にわたるイベントを含める（選択年の近辺だけではない）
- その領土の歴史上最も重要なイベントを選ぶ
- 高校世界史で扱われる水準の主要イベントに限定する

### 良いイベント（主要な転換点）
- 国家の建国・成立
- 大規模な戦争・征服・敗北
- 王朝の交代
- 領土の拡大・喪失
- 国家の滅亡・解体

### 悪いイベント（些末または曖昧）
- "経済が発展した" → 曖昧すぎる
- "人口が増加した" → 具体的なイベントではない
- 軽微な行政上の変更
- 一般的な世界史の学習範囲に含まれない局地的な反乱・小規模な戦闘

## profile フィールド

### dynasty フィールド
- 領土名 ≠ 王朝名の場合のみ含める
  - あり: "フランス王国" → dynasty: "ブルボン朝"
  - なし: "アッバース朝" → dynasty を省略（名前自体が王朝名）
  - なし: "唐" → dynasty を省略（名前自体が王朝名）

### leader フィールド
- 日本語で最も一般的に知られている表記を使用する
- 特定に役立つ場合は称号を含める（例: "ルイ14世"、"康熙帝"）
- 高校世界史で一般的に扱われる人物に限定する

### regime フィールド
- 標準的な日本語の政治学用語を使用する
- 例: "絶対王政"、"幕藩体制"、"共和政"、"帝政"、"立憲君主制"、"神権政治"

## 1 領土 × 1 年 = 1 エントリ

各 Notion ページ = 特定の年における 1 つの領土。複数の年に出現する領土は、各年ごとに個別のページが必要。

## 年の選択

GeoJSON データに存在する年を選択する。利用可能な年（-500 以降）:
`-500, -400, -323, -300, -200, -100, -1, 100, 200, 300, 400, 500, 600, 700, 800, 900, 1000, 1100, 1200, 1279, 1300, 1400, 1492, 1500, 1530, 1600, 1650, 1700, 1715, 1783, 1800, 1815, 1880, 1900, 1914, 1920, 1930, 1938, 1945, 1960, 1994, 2000, 2010`

その年に領土が GeoJSON データに存在する場合のみエントリを作成する。
